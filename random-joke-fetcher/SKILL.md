# Random Joke Fetcher Skill

This skill allows you to retrieve a random joke using the Official Joke API. 
Whenever the user asks for a programming joke, general joke, or just wants to laugh, you can execute this skill.

## How to use

Run the provided Node.js script from the root of the project or the skill directory.

```bash
node scripts/fetch-joke.js
```

The script will fetch the joke and output it to the console. You should then share the setup and punchline with the user.
